1. Install greasemonkey addon for Firefox browser
   (Or tampermonkey extension for Chrome browser)
2. Install the script
3. Run the script before access to pluralsight.com
4. Switch to "Transcript" tab, then refresh the browser, wait 5 seconds then there will be a download button
5. Click and download the desired subtitle (.srt format)

Note: you can adjust the time wait (default is 5s) in the script based on your internet speed
